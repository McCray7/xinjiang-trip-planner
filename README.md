# 新疆旅行计划 Web

这是一个手机优先的静态网页，用来比较两套新疆伊犁自驾方案。

## 功能

- 切换 `8天7晚` 和 `6天5晚`
- 查看每日路线、住宿、预计里程和驾驶时长
- 在真实世界地图上高亮当前选中的日程
- 快速比较两套方案
- 查看出发前清单

## 地图

当前版本使用 `Leaflet + OpenStreetMap` 作为真实地图底图，不需要额外 API Key。

距离和驾驶时间保存在本地数据中，定位为旅行规划预估值，用于分享和行程查看，不等同于实时导航结果。

## 本地打开

直接打开 `index.html`，或者用任意静态文件服务器打开整个目录。

## 部署

这个项目可以直接部署到 GitHub Pages 或 Vercel。
